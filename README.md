# Chatbot app
I have made this app using Streamlit and Google's Generative AI API.

To run this app:

- you need to have a [Google API key](https://aistudio.google.com/app/apikey) and save it in a file called `.env` in the same folder as the `app.py` file.
    ```
    API_KEY="Your-API-KEY"
    ```

- install the following packages using pip
    ```
    pip install -r requirements.txt
    ```

- use the terminal command 
    ```
    streamlit run app.py
    python -m streamlit run app.py
    ```

To check the demo [click here](https://chatbot-160626.streamlit.app/)
